# Gmail AI Auto Reply Agent

## Project Overview

This project is an AI-powered email automation workflow built using n8n.

Features:

* Gmail email monitoring
* Email summarization using AI
* Professional reply generation
* Automatic Gmail replies
* Google Sheets logging
* Structured JSON outputs

---

## Workflow

Gmail Trigger
→ Email Extraction
→ AI Summarizer
→ Structured Output Parser
→ AI Reply Generator
→ Gmail Reply
→ Google Sheets

---

## Technologies

* n8n
* Gmail API
* Google Sheets API
* Groq LLM
* JSON
* AI Agents

---

## Installation

1. Import workflow JSON into n8n
2. Configure Gmail credentials
3. Configure Google Sheets credentials
4. Configure Groq API
5. Activate workflow
